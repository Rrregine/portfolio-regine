# 🌐 Portfolio – Regine

🔗 Live Demo: https://portfolio-regine.azurewebsites.net/

---

## 📖 Overview

This is my personal portfolio website built using **C# and ASP.NET MVC**.
It showcases my projects, background, and skills as a developer.

The application follows the **Model-View-Controller (MVC)** architectural pattern, separating the application into different layers for better organization and maintainability.

---

## 🚀 Features

* Multi-page portfolio website
* Structured navigation between pages
* Server-side rendering using ASP.NET MVC
* Clean separation of concerns (Model, View, Controller)
* Deployed and accessible online via Azure

---

## 🛠 Tech Stack

* **Backend:** C#, ASP.NET MVC
* **Frontend:** HTML, CSS
* **Architecture:** MVC pattern
* **Deployment:** Azure App Service

---

## 🧠 Key Highlights

* Designed and implemented a web application using the **MVC architecture**
* Organized code into **Models, Views, and Controllers** for clarity and scalability
* Gained hands-on experience with **server-side web development in C#**
* Successfully deployed a live project using **Microsoft Azure**

---

## 🧩 MVC Structure

* **Model:** Handles application data and business logic
* **View:** Responsible for UI and presentation
* **Controller:** Manages user input and application flow

This separation improves code readability and makes the project easier to maintain and extend.

---

## ☁️ Deployment

The application is deployed on **Azure App Service**, allowing the project to be accessed online and demonstrating basic cloud deployment skills.

---

## 📌 Notes

This project uses a **monolithic architecture**, where frontend, backend, and application logic are integrated into a single ASP.NET MVC application.

This approach helped me understand how different parts of a web application work together in a real-world environment.
